# Sayan_CtrlZ — Python Developer & Data Scientist Portfolio

A high-performance, responsive portfolio showcasing professional software development capabilities in full-stack Python architectures, data science, and AI/ML pipelines. Inspired by high-contrast **Neo-Brutalist** design aesthetics, built with React, Vite, and GSAP.

## 🚀 About Sayan_CtrlZ
* **Location:** Agartala, Tripura, India
* **Education:** B.S. Student in Data Science & Applications at **IIT Madras** (2nd Year)
* **Core Specialty:** Full-Stack Python, Backend Engineering, and Data Science / Machine Learning

### 🐍 Backend & AI/ML
* **Languages:** Python, JavaScript, TypeScript, SQL

* **Databases:** PostgreSQL, MySQL, SQLite

---

## ✨ Features
* **Neo-Brutalist Visual Identity:** Crisp high-contrast elements, bold typography, offset hard shadows, and geometric grid layouts.
* **Scroll-Linked 360° Rotational Transition:** Custom GSAP scroll triggers that rotate viewport wrapper elements dynamically to reveal sections with 3D momentum.
* **Dynamic Typestate Role Selector:** Responsive, live typing simulator that highlights multiple roles (Python Developer, Data Scientist, Data Analyst, Backend Coder) dynamically.
* **Expandable Grid Matrix:** Toggleable 3-column skills list with lazy node load-outs.
* **Fully Responsive:** Beautiful layout structure fluidly adaptive from small screens up to high-resolution desktop environments.

## ⚙️ Development & Local Setup

### Prerequisites
* **Node.js** (v18 or higher)
* **npm** or **yarn**

### Quickstart

1. **Install Dependencies:**
   ```bash
   npm install
   ```

2. **Run Development Server:**
   ```bash
   npm run dev
   ```

3. **Build & Optimize for Production:**
   ```bash
   npm run build
   ```

4. **Lint and Typecheck:**
   ```bash
   npm run lint
   ```

---

## 🌐 Deployment to Vercel / Cloud Platforms

This project is fully optimized for continuous delivery on **Vercel**, **Render**, or other frontend hosting clouds.

1. Install the Vercel CLI or import the GitHub repository to the Vercel dashboard.
2. Set the build command to `npm run build` and the output directory to `dist`.
3. Deploy and enjoy instantaneous cold-starts.
